# Lesson 15: Impedance Matching — Making Antennas Play Nice

**Unit:** 3 — Design & Construction  
**Prerequisites:** Lessons 8, 11–14  
**Equipment needed:** Components (L/C), ferrite core, VNA  
**Estimated time:** Theory 30 min + Lab 60 min

---

## Learning Objectives

- Design an L-network matching circuit for a given impedance mismatch
- Use a Smith chart to plan a matching strategy visually
- Explain what a balun does and when you need one
- Build and VNA-verify a matching network

---

## Sections to Write

### Why Matching Matters — Maximum Power Transfer, Minimum Reflections
### L-Network Matching — Series/Shunt Inductor/Capacitor Combinations
### Smith Chart Matching — Plot Load Z, Add Components to Reach Centre
### Baluns — Balanced to Unbalanced (1:1 Choke, 4:1 Transformer)
### When You Need a Balun (Dipole → Coax = Always)
### Stub Matching for Advanced Applications
### Lab: Measure a Mismatched Antenna, Design L-Network, Build, Re-Measure
### Lab: Wind a 1:1 Choke Balun and Test on Dipole
### Interactive: L-network calculator with Smith chart path visualisation

---

## 🏆 UNIT 3 CAPSTONE: Design-Build-Measure Project

Design an antenna for a frequency of your choosing:
1. Model in software (4nec2 or MMANA-GAL)
2. Build it physically
3. VNA-measure — compare model predictions to reality
4. Iterate (trim, adjust) until SWR < 1.5:1 at target
5. Write up the complete design cycle with before/after data
6. Publish as a build guide on RF-Hub
