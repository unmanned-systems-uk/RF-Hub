Direction Finding with Antenna Arrays

How amplitude comparison, phase measurement, and specialised array geometries locate RF transmitters — from fox hunting to search and rescue.

📚 Unit 4: Advanced Systems ⚡ Prerequisites: Lesson 17 🛠 Equipment: Optional — two antennas + SDR ⏱ 25 minutes Advanced
CHAPTER 01

Learning Objectives

  • Explain basic direction-finding principles: amplitude comparison and phase comparison
  • Describe Watson-Watt, Adcock, and interferometer DF systems
  • Understand the relationship between DF accuracy and array geometry
  • Appreciate practical DF applications: fox hunting, SIGINT, and search and rescue
CHAPTER 02

Amplitude DF — Comparing Signal Strength on Directional Antennas

The simplest direction-finding technique exploits what every antenna has: a non-uniform radiation pattern. A directional antenna (Yagi, loop, dish) receives more signal from the direction it points than from off-axis. By rotating the antenna and observing signal strength, the direction of arrival is found at the orientation where received power is maximum — or minimum.

💡
Null steering is more accurate than peak finding: The pattern null of a loop or Yagi is much sharper than the main lobe peak. Rotating to the null direction gives better angular resolution than rotating to the maximum. For fox hunting, always DF using the null, not the peak.

Amplitude DF has several practical limitations:

  • Requires physical antenna rotation (slow, obvious)
  • Multipath reflections create false bearings (the reflected signal can be stronger than the direct path)
  • The 180° ambiguity: a cardioid or figure-8 pattern has two lobes or nulls — you cannot tell which hemisphere the signal is in without additional information
  • Accuracy: typically ±5–10° for manual Yagi DF, improving to ±2° with careful technique

To resolve the 180° ambiguity, a sense antenna is used — an omnidirectional element whose output is combined with the directional element to produce a cardioid pattern with a single well-defined null.

CHAPTER 03

Phase DF — Measuring Time/Phase Difference of Arrival

Phase comparison DF (also called Time Difference of Arrival, TDOA) uses two or more spatially separated antennas. A signal arriving from angle θ arrives at the two antennas at slightly different times — the nearer antenna receives it first. This time difference corresponds to a phase difference Δφ at the operating frequency.

Interferometer DF — Phase to Angle
Δφ = (2π · d · sin θ) / λ

θ = arcsin(λ · Δφ / (2π · d))

where:
Δφ = measured phase difference between the two antennas
d = baseline length (separation between antennas)
λ = wavelength
θ = angle of arrival from broadside (perpendicular to baseline)

Phase DF is inherently more accurate than amplitude DF because phase can be measured very precisely. The longer the baseline d, the more sensitive the system — a small angle change produces a larger Δφ. However, longer baselines introduce ambiguity: if d > λ/2, multiple angles of arrival produce the same Δφ (the same issue as grating lobes in arrays).

⚠️
Ambiguity resolution: A long-baseline interferometer measures Δφ modulo 2π. Multiple baselines of different lengths are used together — the short baseline (d < λ/2) resolves the integer ambiguity, while the long baseline provides accuracy. This is the same technique used in GPS and VLBI radio astronomy.
CHAPTER 04

Classic DF Systems — Watson-Watt, Adcock, Interferometer

Watson-Watt
Two orthogonal (cross-shaped) loop antennas plus one sense antenna. The two loops produce X/Y components of the signal's direction vector. Their amplitude ratio gives tan(θ). The sense antenna resolves the 180° ambiguity. Outputs a direct bearing with no rotation required.
Pros: No moving parts, fast reading. Cons: Susceptible to sky-wave contamination (vertically polarised sky waves create errors in horizontally polarised loops). Accuracy ~1–3°.
Adcock
Four vertical elements arranged in a cross, wired so horizontally polarised signals cancel and only vertically polarised signals contribute. Effectively a Watson-Watt variant immune to sky-wave interference. Widely used for MF/HF DF where ionospheric propagation dominates.
Pros: Sky-wave rejection. Cons: Physically larger than Watson-Watt loops. Accuracy ~1–2°.
Interferometer
Two or more spaced receivers measuring phase difference of arrival. Baseline length determines sensitivity. Multiple baselines resolve ambiguity. Typically implemented digitally — two SDR channels with a common clock. Modern SIGINT systems use circular arrays of 8–16 elements.
Pros: Very high accuracy (sub-degree). Cons: Phase ambiguity, requires coherent receivers. Accuracy <1° with good hardware.
Doppler DF
A single antenna rapidly switched in a circle creates an apparent Doppler shift that varies sinusoidally with the signal's bearing angle. The phase of the Doppler modulation relative to the switching cycle reveals the bearing. Common in simple hand-held VHF/UHF fox-hunting receivers.
Pros: Cheap, simple, compact. Cons: Lower accuracy, works best on narrowband FM signals. Accuracy ~5°.
CHAPTER 05

Accuracy Factors — Baseline, Frequency, SNR

FactorEffect on accuracyPractical implication
Baseline length d Longer → more sensitive (larger Δφ per degree) Use d = 5–20λ for interferometry, but ambiguity must be resolved
Operating frequency Higher f → shorter λ → given baseline = more phase sensitive UHF DF (433 MHz, 1.2 GHz) is inherently more accurate than HF
SNR Lower SNR → phase measurement noise → larger bearing error Cramer-Rao bound: σ_θ ∝ λ / (d·cos(θ)·SNR)
Multipath Reflected signals create apparent second source Major limitation in urban areas; use wideband or FHSS signals
Array calibration Phase/amplitude errors in receivers and cables create systematic bearing errors Calibration tables or calibration sources are essential for accuracy
Number of snapshots More observations → averaging → better accuracy For a stationary transmitter, integrate over time to improve SNR
CHAPTER 06

Applications — Fox Hunting to SIGINT

🦊

Amateur Fox Hunting

Competitors locate a hidden 2m transmitter using directional antennas. Amplitude null-DF with a Yagi. Multiple bearings from different locations triangulate the fox. Excellent practical training for DF skills.

✈️

Search & Rescue (SAR)

Emergency locator beacons (ELT, PLB, EPIRB) transmit on 406 MHz and 121.5 MHz. SAR aircraft use DF to home on survivors. COSPAS-SARSAT satellites provide initial location; aircraft DF narrows to exact position.

🕵️

SIGINT

Intelligence gathering by locating hostile transmitters. Wide-baseline interferometer networks with widely spaced receive sites (TDOA). Used for locating radar, communications, and missile guidance emitters.

🚔

Spectrum Enforcement

Ofcom uses DF to locate unlicensed pirate radio stations, jammers, and interference sources. Vehicles with rooftop DF arrays triangulate sources while mobile. Fixed DF networks provide permanent coverage of key areas.

🛩️

VOR Navigation

VHF Omnidirectional Range — aircraft DF from ground beacons. The beacon transmits a rotating directional pattern; the phase difference of the received signal vs a reference tells the aircraft its magnetic bearing from the station.

⚓

AIS & Maritime

Automatic Identification System (AIS) tracking uses TDOA across multiple receive stations to locate vessels even when GPS is spoofed. Phase-difference DF provides independent position verification.

CHAPTER 07

Optional Lab — Amplitude DF with Yagi + SDR

🛠
Equipment: 2m Yagi (or any directional VHF/UHF antenna), RTL-SDR or HackRF, SDR# or GQRX, a known FM broadcast transmitter or a handheld transceiver as a beacon.
Goal: Locate the transmitter to within 10° bearing accuracy using null-DF.
01 Set Up the Receiver

Connect the SDR to your directional antenna. Open SDR# or GQRX and tune to your target signal (FM station 87.5–108 MHz, or a 2m beacon at 144.3 MHz). Confirm you can see the signal clearly. Note the S-meter or signal strength indicator value at your current orientation — this is your reference.

02 Find the Rough Peak Direction

Slowly rotate the antenna 360° while watching signal strength. Note the approximate direction of the peak (strongest signal). This gives you the rough bearing — but the peak is broad and hard to pin down accurately. Note ±45° around the peak as the "likely sector".

03 Find the Null for Precision

Rotate 90° from the peak — this is where the Yagi sidelobe null should be (for a well-designed Yagi the pattern is symmetric). Slowly sweep through this region. The null is very sharp — signal strength drops 15–20 dB in just a few degrees. The null direction is perpendicular to the transmitter bearing. Rotate another 90° to get the transmitter bearing. Record the compass heading.

04 Take Bearings from Two Locations

Move to a second location at least 200m from the first (further is better for triangulation accuracy). Repeat steps 2 and 3. Record the compass heading. Draw both bearing lines on a map. Where they intersect is the transmitter location. The accuracy depends on the angle of cut — 90° crossing gives best triangulation.

05 Evaluate and Record

Compare your triangulated position to the known transmitter location (FM transmitter tower, or your beacon operator's location). Calculate the bearing error. Note any sources of error: multipath from buildings, difficulty identifying the exact null, compass accuracy. A result within 10° is good for manual amplitude DF.

Knowledge Check

1. Why is the pattern null preferred over the pattern peak for amplitude DF?

Show answer

The null is much sharper — signal strength changes by many dB over just a few degrees near the null, compared to the broad, flat peak region where strength changes slowly with angle. The sharper the feature, the more precisely you can identify its direction.

2. What is the role of the sense antenna in a Watson-Watt DF system?

Show answer

The sense antenna is an omnidirectional element. Its output is combined with the figure-8 loop pattern. The combination produces a cardioid (heart-shaped) pattern with a single null — resolving the 180° ambiguity that a figure-8 loop alone cannot resolve (a figure-8 has two identical nulls 180° apart).

3. An interferometer uses two antennas separated by d = 3λ. What is the maximum phase difference at θ = 45°?

Show answer

Δφ = (2π × 3λ × sin45°) / λ = 2π × 3 × 0.707 = 6π × 0.707 ≈ 13.3 radians. This is 13.3 / (2π) ≈ 2.1 complete cycles — so there are 2 full ambiguities in the measurement. A shorter baseline (d < λ/2) is needed to remove ambiguity, then the long baseline provides precision.

4. What is the main difference between an Adcock array and a Watson-Watt system?

Show answer

The Adcock array uses vertical elements that suppress horizontally polarised components, making it immune to sky-wave interference (ionospheric reflections arrive at a shallow angle with horizontal polarisation). Watson-Watt uses horizontal loops, which are affected by sky waves. Adcock is preferred for HF DF where ionospheric paths are common.

5. Why does a longer interferometer baseline improve DF accuracy?

Show answer

A longer baseline d produces a larger phase difference Δφ = (2πd sinθ)/λ for the same angle θ. The phase measurement has a fixed noise level, so a larger Δφ means better signal-to-noise in the phase domain. The angular accuracy σ_θ ≈ λ/(2πd·cosθ·SNR), which improves proportionally with d.

✅
Lesson 18 Summary: Direction finding uses antenna pattern differences (amplitude DF) or phase differences of arrival (phase/interferometer DF) to locate transmitters. Watson-Watt and Adcock systems provide bearing without rotation; interferometers provide high accuracy (sub-degree) using phase measurements across spaced antennas. Accuracy improves with longer baselines, higher frequency, and better SNR. Applications range from amateur fox hunting and emergency SAR to spectrum enforcement and military SIGINT.
