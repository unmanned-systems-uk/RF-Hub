Antenna Arrays — Strength in Numbers

How combining multiple antenna elements multiplies gain and shapes radiation patterns without increasing individual element size.

📚 Unit 4: Advanced Systems ⚡ Prerequisites: Unit 3 complete 🛠 Equipment: None (theory) ⏱ 30 minutes Advanced
CHAPTER 01

Learning Objectives

  • Explain how combining multiple antennas increases gain and shapes patterns
  • Calculate the array factor for simple linear arrays
  • Understand the relationship between element spacing and grating lobes
  • Distinguish between broadside and endfire arrays
CHAPTER 02

Why Arrays — More Gain Without Bigger Elements

A single half-wave dipole has a gain of 2.15 dBi and a relatively broad radiation pattern. To get more gain you have two choices: use a more complex individual element (like a Yagi), or use multiple simpler elements working together.

An array achieves gain through constructive interference. Multiple elements radiate simultaneously. In the desired direction the signals add coherently — like the outputs of multiple signal generators being combined in phase. In undesired directions the signals partially cancel. The result: a narrower, stronger main lobe and a suppressed back hemisphere.

This approach has a significant practical advantage: you can scale the system. Four dipoles in an array add approximately 6 dB of gain over a single dipole. Eight dipoles add approximately 9 dB. You are literally buying gain by adding hardware.

💡
Arrays are everywhere: Your mobile phone is connected to a cellular base station using cross-polarised antenna panels that are really small arrays. The 4G/5G "blade" antennas you see on masts contain 8–64 individual radiating elements. Massive MIMO arrays use 128 or more. Even a simple Yagi is a linear array — the difference is that Yagi elements are parasitically coupled rather than directly fed.
CHAPTER 03

Array Factor × Element Pattern = Total Pattern

The radiation pattern of an array is the product of two components:

📐
Total Pattern = Array Factor (AF) × Element Pattern (EP)
The Element Pattern is the pattern a single element would produce in isolation. The Array Factor depends only on the geometry (number of elements, spacing, excitation).

This separation is extremely useful. You can design the Array Factor independently of the element choice. Replace dipoles with patches and the AF is unchanged — only the EP multiplied into it shifts.

For a uniform linear array (ULA) of N elements with equal amplitude and progressive phase shift δ between adjacent elements:

Array Factor — Uniform Linear Array
AF(θ) = sin(Nψ/2) / [N · sin(ψ/2)]

where: ψ = kd·sin(θ) + δ
k = 2π/λ   (wavenumber)
d = element spacing
θ = angle from broadside (perpendicular to array axis)
δ = progressive phase shift between elements

The AF has its main lobe where ψ = 0, and nulls where Nψ/2 = ±π, ±2π, etc. Side lobes fall between the nulls. For large N, the first sidelobe level is approximately −13.3 dB below the main lobe (for uniform excitation).

⚠️
Practical note: The sinc-like AF formula above is the "far-field" form valid when all elements see essentially the same wave direction. It also assumes equal element spacing and identical elements. Real arrays deviate due to mutual coupling, element failures, and manufacturing tolerance.
CHAPTER 04

Linear Arrays — Broadside and Endfire

A linear array has elements arranged along a straight line (the array axis). The geometry of this line determines how the AF relates to physical angle.

📡 Broadside Array

  • δ = 0 (all elements fed in phase)
  • Maximum radiation perpendicular to array axis
  • Collinear dipoles stacked vertically → broadside is horizontal
  • Pattern narrows as N increases
  • Most common type for base station verticals

🔭 Endfire Array

  • δ = −kd (phase leads in the endfire direction)
  • Maximum radiation along the array axis
  • Long Yagi is an endfire array
  • Higher gain but narrower steering range
  • Less common for steerable systems

CHAPTER 05

Element Spacing — λ/2 Rule and Grating Lobes

The spacing between array elements profoundly affects the pattern. The standard choice for a broadside array is d = λ/2, and for good reasons.

Spacing d Visible range of ψ Effect Use case
d < λ/2 Reduced Wider main lobe, lower gain than theoretical, no grating lobes Conformal arrays, physical constraints
d = λ/2 Exactly −π to +π Optimal gain, no grating lobes in ±90° scan, best mutual coupling Standard choice for broadside
λ/2 < d < λ Extended Narrower main lobe but grating lobes appear at edge of visible space Fixed-beam high-gain arrays
d = λ 0 to 2π Grating lobe at θ = 0° and θ = 90° — ambiguous pattern Avoid for most applications

Grating lobes are additional main lobes that appear when element spacing exceeds λ/2. They arise because the array "thinks" it is looking in two directions simultaneously. For a steered array (phased array), grating lobes limit the maximum scan angle — the scan angle must be small enough that no grating lobe enters visible space.

⚠️
The grating lobe condition: For a broadside array with phase steering, grating lobes appear when d·sin(θ_scan) = λ. To scan to ±60° without grating lobes: d < λ / (1 + |sin(60°)|) = λ / 1.866 ≈ 0.536λ. Phased array designers use d ≈ 0.5λ as a safe rule.
CHAPTER 06

Stacking vs Baying — Vertical vs Horizontal Combining

When arrays are described in real-world systems (especially base stations and repeater antennas), two combining strategies are common:

📶 Stacking (Vertical)

  • Elements arranged one above the other
  • Narrows the elevation beamwidth
  • Increases gain in horizontal plane
  • Standard for omnidirectional base stations
  • 4-stack dipoles → ~6 dB gain, ~10° elevation beamwidth
  • Physical height required = N × d (≈ Nλ/2 for λ/2 spacing)

📡 Baying (Horizontal)

  • Elements arranged side by side
  • Narrows the azimuth beamwidth
  • Increases gain in vertical plane — useful for point-to-point
  • Less common in base stations (sectors already directional)
  • Two Yagis bayed side by side → 3 dB gain, ~halved azimuth BW
  • Physical width required = N × d

The terms are sometimes combined: a "2-bay, 4-stack" array has 2 columns of 4 elements = 8 elements total, giving approximately 9 dB gain over a single dipole. This is typical of a high-gain VHF/UHF repeater or broadcast antenna.

✅
Gain approximation rule: Every doubling of the number of elements adds approximately 3 dB of gain (assuming ideal combination). N elements → gain ≈ G_element + 10·log₁₀(N). This ignores mutual coupling losses (~0.5–1 dB typical) and feed network losses.

Knowledge Check

1. A 4-element uniform broadside array uses dipole elements. Approximately how much gain does it have over a single dipole?

Show answer

Approximately 6 dB (10·log₁₀(4) ≈ 6.02 dB). This assumes ideal combination with no feed losses or mutual coupling degradation.

2. What is the standard element spacing for a broadside array, and why?

Show answer

λ/2. At this spacing, the array factor covers exactly the visible angular range (−90° to +90°) without ambiguity, and grating lobes cannot appear for a fixed broadside beam. It also represents a good balance of mutual coupling between elements.

3. What are grating lobes, and when do they appear?

Show answer

Grating lobes are additional main lobes in the radiation pattern that appear when element spacing exceeds approximately λ/2 for broadside arrays, or when a steered beam is scanned too far from broadside. They are undesirable because they mean the antenna radiates (or receives) in an unintended direction with the same gain as the desired direction.

4. What is the difference between "stacking" and "baying" an array?

Show answer

Stacking places elements vertically (one above the other), which narrows the elevation beamwidth and adds gain in the horizontal plane. Baying places elements horizontally (side by side), which narrows the azimuth beamwidth. Both add approximately 3 dB per doubling of elements.

5. For an endfire array, what is the progressive phase shift δ between elements?

Show answer

δ = −kd = −2πd/λ. This phase advance causes the array factor maximum to align with the array axis (θ = 90° from broadside, or directly "endfire"). For λ/2 spacing, δ = −π (180°).

✅
Lesson 16 Summary: Antenna arrays multiply gain by combining multiple elements whose signals interfere constructively in the desired direction and cancel elsewhere. The total pattern equals the array factor multiplied by the element pattern. Standard broadside arrays use λ/2 spacing to avoid grating lobes. Stacking elements narrows the elevation beam; baying narrows the azimuth beam. Each doubling of elements adds approximately 3 dB of gain.
