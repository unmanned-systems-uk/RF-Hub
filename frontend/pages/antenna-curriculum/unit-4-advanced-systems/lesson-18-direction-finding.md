# Lesson 18: Direction Finding with Antenna Arrays

**Unit:** 4 — Advanced Systems  
**Prerequisites:** Lesson 17  
**Equipment needed:** Optional — two antennas + SDR for practical exercise  
**Estimated time:** 25 minutes

---

## Learning Objectives

- Explain basic DF principles (amplitude comparison, phase comparison)
- Describe Watson-Watt, Adcock, and interferometer DF systems
- Understand the relationship between DF accuracy and array geometry
- Appreciate practical DF applications (fox hunting, SIGINT, SAR)

### Applications — Fox Hunting, Signal Intelligence, Search & Rescue
### Optional Lab: Amplitude DF with Yagi + FM Transmitter
### Interactive: DF simulator — place hidden transmitter, use virtual antennas
