Unit 4 · Lesson 19

Practical Phased Arrays — Build and Test

From theory to hardware: element selection, feed networks, phase-shifting techniques, and verifying beam behaviour with an SDR.

📚 Unit 4: Advanced Systems ⚡ Prerequisites: Lessons 16 & 17 🛠 Equipment: SDR, coax, connectors, optional power divider ⏱ 45 minutes Advanced
CHAPTER 01

Learning Objectives

  • Select elements and calculate spacing for a practical phased array
  • Design a simple feed network using a power divider or hybrid coupler
  • Understand how phase is shifted in hardware — delay lines, PIN diodes, digital phase shifters
  • Connect a 2- or 4-element array to an SDR and verify beam behaviour
  • Diagnose common practical problems: imbalance, mutual coupling, feed errors
CHAPTER 02

From Theory to Hardware — What You Actually Need

Theory tells you to apply phase shifts φ = (2πd sinθ)/λ to each element. Hardware must actually implement this. A practical phased array needs five things:

#ComponentFunctionPractical options
1Array elementsRadiate/receiveDipoles, patches, Yagi driven elements
2Power splitter/combinerDivide transmitter power (or combine received signals) equallyWilkinson divider, T-junction, hybrid coupler
3Phase shiftersApply the required phase advance/delay per elementCoax delay lines, PIN diode switches, digital ICs
4Common mode suppressionPrevent shield currents corrupting the patternChoke baluns at each element feedpoint
5Controller/receiverSelect phase states (Tx) or combine signals (Rx)SDR (Rx), switches, FPGA, microcontroller
💡
Starting point recommendation: Build a 2-element receive-only phased array connected to an RTL-SDR or HackRF. This is the simplest possible system and teaches every lesson applicable to much larger arrays. A 2-element array can only steer between two states or form one null — but the principles are identical to a 64-element system.
CHAPTER 03

Element Choice — Dipole, Yagi Segment, or Patch

Half-wave Dipole (recommended for first build)

  • Simple, predictable pattern (omnidirectional in azimuth)
  • 73Ω feedpoint impedance — match to 50Ω with a simple L-network or use folded dipole (288Ω → transformer)
  • Broad element pattern → array factor dominates; errors more obvious
  • At 145 MHz: arm length = 491 mm (λ/4 with 0.95 VF)
  • Balance requirement: use a choke balun (8 turns RG-58 through FT-82-61)

Patch Antenna (higher performance)

  • Low profile, easily mounted flat on a ground plane
  • Inherently single-polarisation, ~8 dBi gain per element
  • ~60° beamwidth → reduces mutual coupling in dense arrays
  • Bandwidth ~3–5% — frequency-selective, may need retuning
  • Easier to maintain consistent spacing on a PCB or plate
  • Common in 433 MHz, 2.4 GHz, and 5.8 GHz arrays
CHAPTER 04

Calculating Spacing — λ/2 and When to Break the Rule

Standard λ/2 Spacing for Common Frequencies
At 145 MHz: λ = 2.067 m → d = 1.034 m (103.4 cm)
At 433 MHz: λ = 0.692 m → d = 0.346 m ( 34.6 cm)
At 868 MHz: λ = 0.345 m → d = 0.173 m ( 17.3 cm)
At 1575 MHz: λ = 0.190 m → d = 0.095 m ( 9.5 cm) [GPS L1]
At 2.4 GHz: λ = 0.125 m → d = 0.063 m ( 6.3 cm)

Use d < λ/2 if you need to scan beyond ±60° without grating lobes. Use d > λ/2 only for fixed-beam high-gain arrays where the beam never steers away from broadside, and you accept the narrowing main lobe.

Physical element size vs spacing: At lower frequencies the element itself (dipole arm length ≈ λ/4) approaches the spacing. This forces mutual coupling — the elements are electromagnetically close. Mutual coupling changes the element impedance and affects the beam pattern. Use an EM simulator (4nec2, MMANA-GAL, or CST) to verify the actual pattern when elements are closely spaced.

CHAPTER 05

Feed Network — Wilkinson Divider, T-Junction, Hybrid Coupler

T-Junction Splitter (simplest)

  • Coax T-connector — completely passive, no matching
  • Two 50Ω ports in parallel → 25Ω at junction — mismatch!
  • Use with two 100Ω elements (e.g., folded dipoles) for a match
  • No isolation between output ports — imbalance in one element affects the other
  • Good for testing; not ideal for precision arrays

Wilkinson Power Divider (recommended)

  • Two λ/4 transformer lines (70.7Ω for equal split) + 100Ω isolation resistor
  • Equal amplitude and phase at both outputs
  • Ports are isolated — load changes on one port don't disturb the other
  • Can be made from coax: cut two λ/4 lengths of 70.7Ω coax (or RG-179 at ~75Ω)
  • Available as a ready-made PCB component or SMA module
💡
Hybrid coupler for 90° phase arrays: A 90° hybrid coupler (branch-line coupler) divides power equally but with a fixed 90° phase relationship between the two output ports. This is ideal for a 2-element array steered to 30° at λ/2 spacing — the 90° phase shift corresponds to θ = 30° exactly. No separate phase shifter needed.
CHAPTER 06

Phase Shifting Hardware — Delay Lines, PIN Diodes, Digital ICs

Coax delay line Fixed length of coaxial cable. Each 1° of phase = λ/(360×VF) metres. Switched in/out with relays. Discrete, depends on line lengths provided Very low — a few metres of coax and relays

CHAPTER 07

Connecting to the SDR — Impedance, Dynamic Range, Common Mode

Connecting multiple array elements to a single-input SDR requires a combiner. The same Wilkinson divider works in reverse as a combiner — but received signals from the elements must be combined, not their sources. Key considerations:

⚠️
Common-mode currents: Shield currents on coaxial cables can flow back along the outer conductor, effectively making the cable itself part of the antenna and corrupting the pattern. Fit a choke balun (5–10 turns of coax through a ferrite toroid) immediately at each element feedpoint. Without baluns, the array pattern will be unpredictable.

Dynamic range: RTL-SDR dongles have limited dynamic range (~50 dB typical). If a very strong local signal is present (FM broadcast, PMR446, nearby transmitter), it may saturate the ADC. Use an attenuator or bandpass filter before the SDR input to protect it. The HackRF has similar limitations.

Cable length matching: All cable runs from each element to the combiner must be of equal electrical length (matched to the wavelength). Even a 10 mm difference in RG-213 at 145 MHz causes a phase error of approximately 10 × 360 / 1364 mm ≈ 2.6°. This systematic error offsets your beam. Use a VNA to verify electrical lengths before assembly.

CHAPTER 08

Verification — Rotating Transmitter Test and Null Steering

Once assembled, verify that the array behaves as predicted. Three tests confirm correct operation:

V1 Rotating Transmitter Test

With the array connected to your SDR and set to broadside (δ = 0°, all equal phase), walk a CW transmitter in a full circle around the array at fixed distance (~10–20m). Plot S-meter reading vs angle. Compare to the theoretical AF pattern. Both should show maximum in the broadside direction and nulls at approximately ±90° off broadside.

V2 Null Steering Check

Place a transmitter at a known angle (e.g., 30° from broadside). Calculate the required phase shift for a null at 30° using φ = (2πd sin30°)/λ. Apply the phase shift. Verify that the signal drops to the expected null depth. For a 2-element array with equal amplitudes, the null depth is theoretically infinite — practical nulls are typically 20–35 dB deep.

V3 Waterfall Evidence

Use SDR# or GQRX waterfall display. With the phase shifter in the broadside state, observe a known signal at a strong level. Switch to the null-steering phase state. You should see the signal fade visibly on the waterfall — a direct visual confirmation that the phased array is functioning. Record the before/after signal level.

Broadside received level: _______ dBm
Null state received level: _______ dBm
Null depth achieved: _______ dB (target: >15 dB for a practical 2-element array)
CHAPTER 09

Common Problems and Fixes

Amplitude Imbalance
Null never goes below −10 dB. Pattern asymmetric. One element reads lower signal.
Check cable lengths with VNA or signal source. Verify both elements are the same height/environment. Check splitter/combiner insertion loss balance.
Phase Error (Wrong Null Angle)
Null occurs at a different angle than calculated. Broadside offset.
Measure cable electrical lengths — trim longer cable to match. Check delay line length calculation (remember velocity factor). VNA measurement of phase vs frequency on each cable run.
Common-Mode Currents
Pattern doesn't match theory at all — broad and asymmetric. Adding/removing hands changes received level.
Add choke baluns at every element feedpoint. Ensure coax outer conductors don't form a large loop. Use shielded enclosures for combiner/phase shifter.
Mutual Coupling
Element impedance off from design value (VNA shows SWR > 2). Pattern distorted near endfire.
Expected for closely-spaced elements. Model in 4nec2 to see the actual coupled impedance. Use an L-network matching at each element feedpoint to restore 50Ω.
Connector Phase Errors
Repeatable phase anomaly that doesn't match cable calculations.
SMA/BNC connector delay ≈ 0.5–2 ns depending on type. At 145 MHz this is 0.07–0.29 degrees — usually negligible. At higher frequencies (433 MHz+) more significant. Use same connector type throughout to keep errors consistent.
CHAPTER 10

Practical Lab — 2-Element Dipole Array + RTL-SDR: Steer a Null

🛠
Materials needed: 2× half-wave dipoles at 145 MHz (or 433 MHz), 2× short coax runs + choke baluns, 1× Wilkinson power combiner (purchased or homebuilt), 2× coax delay lines for phase states (0° and 90° at your frequency), 1× RTL-SDR or HackRF, SDR# or GQRX software, a portable transmitter at your chosen frequency (HT radio, beacon generator, or SDR in Tx mode if HackRF).
01 Build and Align the Elements

Mount two dipoles parallel to each other, separated by λ/2 (103 cm at 145 MHz, or 34.6 cm at 433 MHz). Align both on the same axis (same height, same orientation). Use a horizontal support to maintain accurate spacing. Fit choke baluns at each feedpoint. Measure the SWR of each element individually — target SWR <1.5:1. Record the measured resonant frequency.

Element 1 resonant freq: _______ MHz SWR at target freq: _______
Element 2 resonant freq: _______ MHz SWR at target freq: _______
Element spacing measured: _______ mm (target: _______ mm)
02 Build the Combiner and Equalise Cable Lengths

Connect equal-length coax runs from each element to the Wilkinson combiner. The cable lengths must match electrically. Cut to the same physical length (this is sufficient if using the same cable type from the same reel). Connect the combiner output to the RTL-SDR. Use a reference signal (FM broadcast or known transmitter) to confirm the system receives on all elements.

03 Calculate and Cut the 90° Delay Line

Calculate the physical length of a 90° delay line for your frequency and chosen cable type. Example: at 145 MHz using RG-58 (VF = 0.66), length = (2.067 × 0.66) / 4 = 341 mm. Cut a pigtail of this length. This pigtail is inserted in series with one element's feed cable to shift its phase by 90°, steering the broadside null to 30°.

Calculated delay line length: _______ mm
Cable type/VF used: _______ / _______
Phase shift this provides: 90° at _______ MHz
04 Position Transmitter and Compare Broadside vs Null-Steered

Place a transmitter approximately 30° from the array broadside at a distance of 20–50m. Open SDR# and note signal level with both elements connected and no delay line (broadside state). Insert the 90° delay line in series with the appropriate element. Observe the signal level change. The signal should drop by ≥15 dB as the null steers toward the transmitter.

Transmitter frequency: _______ MHz
Transmitter bearing from array: _______ ° (measured)
Broadside state signal level: _______ dBm
Null-steered signal level: _______ dBm
Null depth achieved: _______ dB
05 Document and Evaluate

Compare your measured null depth to the theoretical maximum (dependent on amplitude balance). If null depth is poor (<15 dB), check for amplitude imbalance between the two signal paths using a signal generator and power meter (or SDR). Adjust cable lengths or add a fixed attenuator in the stronger path to equalise amplitudes. A well-balanced 2-element array should achieve 25–35 dB null depth.

Knowledge Check

1. What is the physical length of a 90° delay line at 433 MHz using RG-213 (VF = 0.66)?

Show answer

λ at 433 MHz = 0.692 m. λ_cable = 0.692 × 0.66 = 0.457 m. 90° = λ_cable/4 = 0.457/4 = 114 mm of RG-213.

2. Why must all cable runs from elements to the combiner be the same electrical length?

Show answer

Any difference in electrical length adds an unintended phase offset to the signals from that element. This offsets the entire beam from the desired steering angle. A 50mm difference in RG-213 at 145 MHz causes a phase error of 50/1364 × 360° ≈ 13°, significantly displacing the null from its target.

3. A 2-element broadside array with perfect amplitude balance can theoretically achieve how deep a null?

Show answer

Theoretically infinite (−∞ dB), because with equal amplitude and exactly 180° phase difference between the two signal paths, they cancel completely. In practice, amplitude imbalance limits this — 1 dB imbalance limits null depth to about −26 dB, 0.5 dB imbalance to about −32 dB.

4. What is the advantage of a Wilkinson power divider over a simple T-junction for array feed networks?

Show answer

The Wilkinson divider provides isolation between the output ports. In a T-junction, a mismatch (or mutual coupling effect) on one element directly affects the signal at the other element's port. The Wilkinson's isolation resistor prevents this — changes in one element's load impedance don't perturb the other element's excitation.

5. You observe that your 2-element null is only 12 dB deep. What is the most likely cause?

Show answer

Amplitude imbalance between the two element signal paths. −12 dB null depth corresponds to approximately 2 dB amplitude difference between the two paths. Check cable insertion losses, splitter port imbalance, and whether both elements have the same SWR (indicating the same impedance and hence same coupling efficiency). Add a fixed attenuator to the stronger path to equalise.

✅
Lesson 19 Summary: Building a practical phased array requires five components: radiating elements, a power splitter/combiner, phase shifters, common-mode suppression (baluns), and a controller/receiver. Standard λ/2 spacing applies regardless of element type. Wilkinson power dividers provide better port isolation than T-junctions. Phase shifting can be implemented with fixed coax delay lines (simplest), PIN diode switches, or digital phase shifter ICs. The most common failure modes are amplitude imbalance and common-mode currents — address these with baluns and careful cable length matching.
