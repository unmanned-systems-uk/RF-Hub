Phased Arrays — Steering the Beam

How controlled phase shifts between array elements steer a beam electronically — no moving parts, arbitrary pointing in microseconds.

📚 Unit 4: Advanced Systems ⚡ Prerequisites: Lesson 16 🛠 Equipment: None (theory) ⏱ 30 minutes Advanced
CHAPTER 01

Learning Objectives

  • Explain how phase differences between elements steer the beam
  • Calculate the inter-element phase shift for a given beam direction
  • Understand amplitude tapering for sidelobe control
  • Describe digital beamforming at a conceptual level
CHAPTER 02

Phase Steering — Delayed Excitation = Tilted Wavefront

In a fixed broadside array, all elements are fed with the same phase. The radiated wavefronts from each element add up in the plane perpendicular to the array axis. Now imagine introducing a small time delay between adjacent elements — the earlier-fired element's wavefront has had longer to propagate. The combined wavefront tilts. The beam steers.

This is the fundamental principle of phased array beam steering: by applying a progressive phase shift across the array, the constructive interference peak is moved away from broadside to any desired angle. No mechanical movement is required. The steering is instantaneous (limited only by the speed of signal propagation through the phase shifters).

💡
Analogy — Marching band: Imagine a line of musicians all starting to play at exactly the same time — the sound goes straight ahead (broadside). Now have each musician start playing 10 milliseconds after the one to their left. The "wavefront" of sound tilts to the right. This is phase steering.

The relationship between the physical delay and the steering angle comes from geometry. For two adjacent elements separated by distance d, you want the wavefront to arrive at angle θ from broadside. The path difference is d·sin(θ). At frequency f, this path difference corresponds to a phase shift of:

Phase Shift for Beam Steering
φ = (2π · d · sin θ) / λ

where:
φ = inter-element phase shift (radians)
d = element spacing
θ = desired beam angle from broadside
λ = wavelength at operating frequency

Each element n in the array is fed with a phase of n × φ. Element 0: 0°, Element 1: φ, Element 2: 2φ, etc. This is the progressive phase shift that tilts the beam to angle θ.

CHAPTER 03

Worked Example — Steering a 145 MHz Array to 30°

Consider an 8-element broadside array operating at 145 MHz with λ/2 spacing. We want to steer the main beam to 30° from broadside.

Step-by-Step Calculation

Frequency: f = 145 MHz
Wavelength: λ = c/f = 299,792,458 / 145×10⁶ = 2.067 m
Element spacing: d = λ/2 = 1.034 m
Target angle: θ = 30°
sin(30°) = 0.500
φ = (2π × 1.034 × 0.500) / 2.067 = π/2 = 90°
Element 0: 0° · Element 1: 90° · Element 2: 180° · Element 3: 270° ···
📐
Phase vs time delay: A phase shift of 90° at 145 MHz corresponds to a time delay of 90/(360 × 145×10⁶) = 1.72 ns. This is equivalent to about 345 mm of free-space path or approximately 228 mm of RG-213 cable (velocity factor ≈ 0.66). Hardware phase shifters implement exactly this.

CHAPTER 04

Beam Scanning Range — Practical Limits

Phased arrays can theoretically steer to ±90° from broadside, but practical scan range is significantly limited:

LimitCauseTypical effect
Grating lobes Spacing d ≥ λ creates ambiguous pattern Limit scan to ±(30–60)° for λ/2 spacing depending on requirements
Element pattern roll-off Individual dipole/patch gain falls off at large angles Gain drops ~3 dB at 60°, ~10 dB at 80°
Mutual coupling variation Impedance of each element changes as beam steers Mismatch loss increases at large scan angles
Phase shifter resolution Finite phase step size (e.g., 4-bit = 22.5° steps) Quantisation error → small spurious sidelobes

For most practical phased arrays, a scan range of ±45° to ±60° from broadside is considered achievable with acceptable performance. Wider scan requires active element impedance matching, careful element design, or sub-half-wavelength spacing.

⚠️
The "scan blindness" phenomenon: At certain scan angles, mutual coupling between elements can cause a near-perfect standing wave — the array becomes highly mismatched and effective gain collapses to nearly zero. This is scan blindness. It is predicted by analysing the array's active reflection coefficient as a function of scan angle.
CHAPTER 05

Amplitude Tapering — Controlling Sidelobes

With uniform excitation (all elements at equal amplitude), a linear array produces a first sidelobe approximately −13.3 dB below the main lobe. This is often unacceptable — sidelobes mean the array receives interference from, or radiates into, unintended directions.

Amplitude tapering reduces the sidelobe levels by feeding the outer elements with less power than the central elements. The trade-off is that the main lobe widens slightly and peak gain decreases compared to uniform excitation. The aperture efficiency decreases.

Uniform
−13.3 dB
Highest gain, narrowest beam, highest sidelobes. All elements equal amplitude.
Taylor (−25 dB)
−25 dB
~12 dB sidelobe improvement. ~1 dB gain loss. Commonly used in radar and communications.
Chebyshev (equiripple)
Specified
All sidelobes equal a preset level. Minimum main lobe width for a given SLL. Common in radar.

Common tapering windows borrowed from DSP include: Hamming, Hanning, Blackman, Kaiser-Bessel. These familiar filters are applied not in time but across the spatial aperture of the array.

💡
Array design = filter design: The analogy between antenna arrays and FIR digital filters is exact. Array element positions and amplitudes correspond to filter tap positions and coefficients. Time → space, temporal frequency → spatial frequency (sin θ). You can design arrays using digital filter design tools.
CHAPTER 06

Digital Beamforming Preview — Phase Shifts in Software

Traditional phased arrays apply phase shifts in analogue hardware (phase shifters, delay lines). Digital beamforming (DBF) takes a different approach: every array element has its own receiver that digitises the received signal to I/Q samples. The phase shifts (and amplitude weights) are then applied in software — multiplication of complex numbers.

This has transformative consequences:

Knowledge Check

1. A 4-element array at 433 MHz uses λ/2 spacing. What inter-element phase shift is needed to steer the beam to 20° from broadside?

Show answer

λ = c/f = 0.692 m. d = λ/2 = 0.346 m. φ = (2π × 0.346 × sin20°) / 0.692 = (2π × 0.346 × 0.342) / 0.692 = 2π × 0.171 = 1.075 rad = 61.6°.

2. Why does amplitude tapering reduce sidelobes at the cost of a wider main lobe?

Show answer

Reducing the amplitude of outer elements is equivalent to windowing the aperture. The Fourier transform of the windowed aperture has a wider main lobe (lower spatial resolution) but lower sidelobes. The trade-off is exact: there is no free lunch — you cannot reduce both simultaneously for a given aperture size.

3. What is the key advantage of digital beamforming over analogue phase shifting?

Show answer

Multiple simultaneous beams can be formed from the same received data. An analogue array can only point in one direction at a time (one set of phase shifter states). A digital beamformer can simultaneously form beams to multiple directions by applying different weight vectors to the same digitised element data, enabling MU-MIMO (multi-user MIMO) in 5G systems.

4. What is "scan blindness" in a phased array?

Show answer

Scan blindness is a phenomenon where at a specific scan angle, mutual coupling between array elements causes a near-total impedance mismatch. The array effectively stops radiating/receiving at that angle. It is a resonance effect of the periodic element structure and must be predicted during design.

5. A uniform 8-element array has a first sidelobe at −13.3 dB. You redesign with Taylor tapering giving −25 dB sidelobes. What is the approximate gain penalty?

Show answer

Approximately 1–1.5 dB reduction in peak gain (aperture efficiency falls from 1.0 for uniform to ~0.75–0.8 for Taylor −25 dB). The main lobe also widens by approximately 20–30%.

✅
Lesson 17 Summary: Phased array beam steering works by applying a progressive phase shift φ = (2πd sinθ)/λ across array elements, tilting the interference maximum to angle θ. The scan range is practically limited to ±45–60° by grating lobes, element pattern roll-off, and mutual coupling effects. Amplitude tapering trades main lobe width for reduced sidelobe levels. Digital beamforming replaces hardware phase shifters with complex weighting in DSP, enabling simultaneous multiple beams and adaptive nulling.
