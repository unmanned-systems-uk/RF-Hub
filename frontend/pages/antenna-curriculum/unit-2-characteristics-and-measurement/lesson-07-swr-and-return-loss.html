SWR and Return Loss: Is My Antenna Working?

Reflection coefficient, standing wave ratio, return loss — and what to do about a mismatch

Unit 2 — Characteristics & Measurement Prerequisites: Lesson 6 (Impedance) Equipment: None ⏱ 25 min Intermediate
CHAPTER 01

Learning Objectives

By the end of this lesson you will be able to:

  • Define SWR, return loss, and reflection coefficient (Γ)
  • Convert between SWR, return loss, and reflection coefficient
  • Set practical SWR thresholds for different applications
  • Explain what actually happens to reflected power
CHAPTER 02

The Reflection Coefficient (Γ)

💡
The Big Idea: In Lesson 6 we learned that impedance mismatch means the antenna doesn't "look like" 50 Ω to the feedline. When impedances don't match, some of the forward-travelling RF power reflects back toward the transmitter. The reflection coefficient quantifies exactly how much.

The reflection coefficient (Γ, Greek capital Gamma) is the fundamental quantity from which SWR and return loss are both derived:

Γ = (Z_load − Z₀) / (Z_load + Z₀)

where:
  Z_load = antenna feedpoint impedance (complex: R + jX)
  Z₀ = feedline characteristic impedance (usually 50 Ω)
  |Γ| ranges from 0 (perfect match) to 1 (total reflection)

Examples:

  • Antenna Z = 50 Ω: Γ = (50−50)/(50+50) = 0 — perfect match, no reflection
  • Antenna Z = 100 Ω: Γ = (100−50)/(100+50) = 0.33 — some reflection
  • Antenna Z = open circuit: Γ = 1 — total reflection
  • Antenna Z = short circuit: Γ = −1 — total reflection, opposite phase
🔑
S11 on a VNA is Γ. When your VNA displays S11, it is measuring the complex reflection coefficient at port 1. Every SWR and return loss value your VNA shows is derived from this single underlying measurement.
CHAPTER 03

SWR: Standing Wave Ratio

When forward and reflected waves coexist on a feedline, they interfere and create a standing wave — a stationary pattern of voltage maxima and minima along the cable. The Standing Wave Ratio (SWR) describes how pronounced this pattern is:

SWR = (1 + |Γ|) / (1 − |Γ|)

Ranges from 1:1 (perfect match, no standing wave) to ∞:1 (total reflection)

Practical SWR Thresholds

1.0–1.5 : 1
Excellent — deploy without concern. Well under 4% power reflected.
1.5–2.0 : 1
Good — acceptable for most applications. 11% power reflected at 2:1.
2.0–3.0 : 1
Marginal — investigate and fix if possible. Increased feedline loss.

For receive-only SDR work, high SWR matters less — you lose some signal but won't damage equipment. For transmitting, sustained high SWR can cause feedline heating and stress the transmitter's output stage, though modern radios have built-in protection.

CHAPTER 04

Return Loss

Return loss is simply Γ expressed in decibels. It describes how much the reflected signal is attenuated compared to the forward signal:

Return Loss (dB) = −20 × log₁₀(|Γ|)

A higher (less negative) return loss number = better match
  |Γ| = 0.01 → RL = 40 dB (excellent)
  |Γ| = 0.1  → RL = 20 dB (good)
  |Γ| = 0.33 → RL = 9.5 dB (marginal)
📊
Note on sign convention: VNA displays often show S11 as a negative number (e.g., −15 dB). The magnitude of this number is the return loss. A reading of −20 dB means the return loss is 20 dB — which is good. The more negative, the better the match.
CHAPTER 05

Conversion Reference Table

These three quantities are three faces of the same measurement. Use this table to convert between them:

2.0 : 1 0.333 9.5 dB 11% Acceptable

5.0 : 1 0.667 3.5 dB 44% Poor
10.0 : 1 0.818 1.7 dB 67% Very poor
∞ : 1 1.000 0 dB 100% Total mismatch
CHAPTER 06

What Happens to Reflected Power?

⚠️
Common myth: "reflected power damages the transmitter." This is broadly untrue — the real story is more nuanced.

When power reflects from the antenna back toward the transmitter, it reaches the transmitter output. There, it encounters another impedance mismatch (between the feedline and the transmitter's output circuit) and most of it reflects again — this time back toward the antenna. This double-reflection means the power bounces back and forth, eventually being absorbed (as useful radiated signal or feedline heat) rather than entering the transmitter's electronics.

The real concerns with high SWR are:

  • Increased feedline loss: Each trip down and back the cable dissipates energy. A lossy cable at SWR 3:1 wastes significantly more power than at 1:1. This is the primary practical problem.
  • Voltage standing wave peaks: High SWR creates voltage maxima along the cable that can exceed cable voltage rating, potentially arcing in connectors or coax.
  • Transmitter stress: Solid-state transmitters detect high SWR and reduce power to protect output transistors. This is protection, not damage — but it means reduced output power.
✅
Bottom line: For receive-only work (SDR), SWR doesn't damage anything — it just reduces signal. For transmitting, aim for SWR < 2:1, use good-quality low-loss coax, and let the radio's SWR protection circuit do its job.
