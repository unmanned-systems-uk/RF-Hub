Curriculum Overview

The RF-Hub curriculum is organised into four progressive phases. Each phase builds on the last, taking you from first principles through to hands-on measurement and advanced signal systems. Modules are self-contained but designed to be completed in order within each phase.

Phase 1 — Foundations

Core RF theory: electromagnetic waves, frequency and wavelength, antenna fundamentals, transmission lines, and impedance matching. Suitable for complete beginners.

Start → Lesson 1: EM Radiation

Phase 2 — Practical RF

Hands-on skills: SDR operation, spectrum analysis, modulation schemes, propagation, and real-world signal identification using your hardware.

Start → Lesson 6: Impedance

Phase 3 — Measurement

VNA operation, S-parameter measurement, return loss, SWR, Smith chart interpretation, antenna characterisation, and filter analysis.

Start → Lesson 11: Dipole Deep Dive

Phase 4 — Advanced Systems

Phased arrays, direction finding, software beamforming, satellite links, EMC principles, and research-grade experimentation.

Start → Lesson 16: Antenna Arrays
