# Lesson 19: Practical Phased Arrays — Build and Test

**Unit:** 4 — Advanced Systems
**Prerequisites:** Lessons 16 and 17
**Equipment needed:** SDR (RTL-SDR or HackRF), coax, connectors, optional power divider
**Estimated time:** 45 minutes

---

## Learning Objectives

- Select elements and calculate spacing for a practical phased array
- Design a simple feed network using a power divider or hybrid coupler
- Understand how phase is shifted in hardware (delay lines, switched capacitors, digital phase shifters)
- Connect a 2- or 4-element array to an SDR and verify beam behaviour
- Diagnose common practical problems (imbalance, mutual coupling, feed errors)
